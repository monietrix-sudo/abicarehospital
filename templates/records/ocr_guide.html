What is OCR and How AbiCare Uses It

OCR stands for Optical Character Recognition. It reads text from scanned images and PDFs — the same way a human reads a printed page, but automatically.

In AbiCare, you can upload a scanned patient document (e.g. a referral letter, old lab result, or handwritten notes as a photo), click Extract Text with OCR, and the system reads the text out of the image and saves it into the record body — making it searchable and readable without opening the file.

AbiCare uses Tesseract — a free, open-source OCR engine developed by Google. It runs entirely on your server — no internet connection needed, no API fees, no data leaving your system.
1

Install Tesseract on Your Server

Tesseract is the OCR engine. You need to install it at the system level — it is not a Python package.

Ubuntu / Debian / TrueHost VPS
sudo apt update
sudo apt install -y tesseract-ocr tesseract-ocr-eng
# For Yoruba / Hausa / Igbo (if scanning Nigerian documents):
sudo apt install -y tesseract-ocr-yor tesseract-ocr-hau
macOS
brew install tesseract
Windows
Download the installer from: github.com/UB-Mannheim/tesseract/wiki
Install it, then note the install path (usually C:\Program Files\Tesseract-OCR\tesseract.exe). Add this to your .env file:
TESSERACT_CMD=C:\Program Files\Tesseract-OCR\tesseract.exe
Verify installation: Open a terminal and run tesseract --version If you see a version number, it is installed correctly.
2

Install Python Packages

pip install pytesseract Pillow pdf2image
  • pytesseract — Python wrapper that calls Tesseract
  • Pillow — reads image files (JPG, PNG, etc.)
  • pdf2image — converts PDF pages to images before OCR
pdf2image also needs poppler installed on the system for PDF support:
sudo apt install -y poppler-utils (Ubuntu) or brew install poppler (macOS)
3

How to Use OCR in AbiCare — Step by Step

1
Open a patient's Medical Records
Go to any patient profile → Click Upload Record in the header buttons
2
Upload a scanned document
Upload a scanned image (JPG/PNG) or a scanned PDF. Supported: referral letters, old lab results, prescriptions, handwritten notes photographed clearly.
3
Open the record and click Extract Text
Inside the record detail page, click the 🔍 Extract Text with OCR button. The system runs Tesseract on the file and fills the record body with the extracted text.
4
Review and correct the extracted text
OCR is not perfect — especially with handwriting or low quality scans. Click Edit Record to fix any misread words. A version snapshot is automatically saved before OCR so you can always undo.

Tips for Best OCR Results

What works well
  • Typed/printed documents (referral letters, lab forms)
  • Clear, high-resolution scans (300 DPI or above)
  • Good lighting if using a phone camera
  • Flat documents (no wrinkles, no shadows)
  • Black text on white background
What works poorly
  • Handwritten notes (especially rushed doctor handwriting)
  • Very low resolution photos (blurry)
  • Documents photographed at an angle
  • Coloured paper or coloured text
  • Documents with heavy borders or logos obscuring text

Quick Install (Copy & Paste)

Run on your TrueHost VPS:
sudo apt update
sudo apt install -y \
  tesseract-ocr \
  tesseract-ocr-eng \
  poppler-utils

pip install pytesseract \
  Pillow \
  pdf2image
Verify it works:
tesseract --version
python -c "import pytesseract; print('OK')"
After installing, restart the server:
systemctl restart abicare
