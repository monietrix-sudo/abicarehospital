Enabling Automatic Reminders

Run this command every 15 minutes to check for overdue doses and send appointment reminders:

python manage.py send_reminders
Windows Task Scheduler: Create a basic task, trigger every 15 minutes, action = start program → python → with argument manage.py send_reminders

Linux/Mac crontab:
*/15 * * * * cd /path/to/abicare_ehr && python manage.py send_reminders
